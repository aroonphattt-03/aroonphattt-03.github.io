Cache-Manager

A part of the I/O Manager that improves a computer's performance by temporarily storing files in memory instead of reading and writing them to the hard disk. The Cache Manager uses virtual address space to cache data.

GPI: เป็นส่วนหนึ่งของ I/O Manager ที่ช่วยเพิ่มประสิทธิภาพคอมพิวเตอร์โดยการเก็บไฟล์ไว้ชั่วคราวในหน่วยความจำ แทนที่จะอ่านหรือเขียนลงฮาร์ดดิสก์ Cache Manager ใช้พื้นที่ที่อยู่เสมือน (virtual address space) ในการเก็บข้อมูลชั่วคราว.

Gemini: ส่วนหนึ่งของ I/O Manager (ตัวจัดการอินพุต/เอาต์พุต) ที่ช่วยปรับปรุงประสิทธิภาพของคอมพิวเตอร์โดยการจัดเก็บไฟล์ไว้ในหน่วยความจำชั่วคราว แทนที่จะอ่านและเขียนไฟล์ไปยังฮาร์ดดิสก์

ความหมายที่ตัวเราเข้าใจ: ส่วนหนึ่งของคอมพิวเตอร์ที่ช่วยให้ทำงานเร็วขึ้น โดยเก็บข้อมูลไว้ชั่วคราวในหน่วยความจำ แทนที่จะไปอ่านหรือเขียนจากฮาร์ดดิสก์ทุกครั้ง.
